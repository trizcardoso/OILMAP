# OILMAP
A exploração de petróleo ainda sofre com dados geoespaciais dispersos e difíceis de integrar, o que gera retrabalho e atrasa decisões. A Petrobrás utiliza métodos tradicionais que limitam a eficiência. Propõe-se, portanto, uma plataforma web que unifique e analise esses dados, aumentando a precisão, reduzindo custos e agilizando as operações.
